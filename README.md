# Source code dari tutorial CRUD dengan Django dan Bootstrap 4



## Requirement:
- Python versi 3.8 atau di atasnya
- PIP

## Cara Instalasi:
```
pip install -r requirements.txt
```

## Cara menjalankan Migrasi
```
./manage.py migrate
```

## Cara menjalankan aplikasi
```
./manage.py runserver
```
